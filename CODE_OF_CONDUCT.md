# Code of Conduct

## Nuestra promesa
Nos comprometemos a crear un entorno inclusivo, seguro y colaborativo.

## Comportamiento esperado
- Respeto y profesionalismo.
- Colaboración abierta y constructiva.
- Escucha activa y comunicación clara.

## Comportamiento inaceptable
- Lenguaje ofensivo, discriminatorio o agresivo.
- Acoso, intimidación o exclusión deliberada.
- Publicación de información sensible sin permiso.

## Cumplimiento
Los mantenedores pueden aplicar acciones correctivas para proteger la comunidad.
